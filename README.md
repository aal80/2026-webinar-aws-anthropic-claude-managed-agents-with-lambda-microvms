# Deploying secure production-ready agents with Claude Managed Agents on AWS

Join AWS and Anthropic to learn how Claude Managed Agents and AWS Lambda MicroVMs work together to give you fast, secure agent deployments without giving up control. Anthropic hosts the agent loop and model; AWS Lambda MicroVMs run your tool calls in sandboxes you own — with your files, packages, and network policies. We'll walk through the architecture, show a live integration, and cover how to get started.
